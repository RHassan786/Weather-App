Weather App

A simple weather app I built to practice JavaScript.

Live Demo

https://rhassan786.github.io/Weather-App/

(best viewed on desktop, mobile responsiveness isn't done yet)

What it does


Search a city and get the current weather - temperature, feels like, humidity, wind speed, and conditions
Click through to see a 7-day forecast
Color coded temperature and weather conditions


What I learned

I was struggling to really understand async JavaScript, so I built this to practice it hands-on instead of just reading about it. Main things I learned:


Chaining two API calls with async/await (city name → coordinates → weather data)
Handling errors properly with try/catch and checking response.ok
Working with nested JSON responses
Updating the page dynamically with JavaScript instead of hardcoding everything


Built with

HTML, CSS, JavaScript, and the Open-Meteo API for weather data.
